Solarcoin Integration API Reference

Welcome to the SolarCoin community! This API is designed for affiliates and large solar producers to file their claims programmatically. If you haven’t already, you’ll need to register your facility and receive an access token prior to claiming.

Terms of Service: https://solarcoin.org/
Request Content-Types: application/json
Response Content-Types: application/json
Schemes: http, https
Version: 1.0.0
